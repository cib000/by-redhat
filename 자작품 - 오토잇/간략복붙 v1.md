# 간략복붙 v1




### 현재버전 : 0.2

---

프로젝트 목적 :

ㄴ 간단하게 메시지를 저장하도록 구성 한 후 사용자화에 기준과 범위를 둠.


<br><br><br>
실행과 동작 :

ㄴ 단독으로 설정을 저장 활용및 사용자화.

ㄴ 문서를 저장 후 재활용하도록 구성됨.

	- 문서저장시 일시가 적용되어 확인/구분을 쉽게 처리 ( 중복저장 방지 )
	
	- 별도의 하위 폴더를 기준하므로 별도 관리가 필요. ( 불필요한 데이터는 삭제 )
	

ㄴ 저장구문 분리 복원.

ㄴ 버튼으로 간단 복사 또는 붙이기 지원.

ㄴ 최근문서를 재활용가능하도록 5개까지 지원. ( 추가 예정 )
