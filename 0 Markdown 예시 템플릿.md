# 문서 제목 (H1)

문서에 대한 **간단한 설명**을 작성합니다.  
이 문서는 마크다운 문법 예시를 포함합니다.

- 작성자: 이름
- 작성일: YYYY-MM-DD
- 목적: 문서 예시 / 설명용 / 학습용

---

## 1. 개요 (H2)

이 섹션에서는 문서의 **전체적인 목적과 범위**를 설명합니다.

마크다운에서  
문단을 나누려면 **빈 줄**이 필요하며,  
줄바꿈을 강제로 하려면 문장 끝에 공백 두 칸을 넣습니다.  

---

## 2. 제목 레벨 예시

# H1 제목
## H2 제목
### H3 제목
#### H4 제목
##### H5 제목
###### H6 제목

<!-- 일반적으로 H1은 문서 제목에만 사용 -->

1. 첫 번째 단계

3. 세 번째 단계

   - 결과 확인
